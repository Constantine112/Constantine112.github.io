前言：

什么是hexo

官网的解释是：A fast, simple & powerful blog framework，而在我的理解里面，这就是一个能让你快速搭建属于自己博客的github插件。

解释了这么多，现在我们来开始搭建属于自己的博客吧。

搭建步骤

  • 拥有自己github仓库
  • 在你的电脑上有git
  • 安装node.js
  • 安装hexo
  • 推送网站
  • 发布文章
  • 个性化设置
  • 其它

github创建属于自己的仓库

登录github，如果没有账户需要自己注册，点击GitHub中的New repository创建新仓库，仓库名应该为：用户名.http://github.io这个用户名使用你的GitHub帐号名称代替，这是固定写法。

安装git

什么是Git ?简单来说Git是开源的分布式版本控制系统，用于敏捷高效地处理项目。我们网站在本地搭建好了，需要使用Git同步到GitHub上。如果想要了解Git的细节，参看廖雪峰老师的Git教程。而git则能从官网下载：Git - Downloading Package 。具体在安装之后的初始化可以查看廖雪峰老师的git教程，里面有详细介绍。
